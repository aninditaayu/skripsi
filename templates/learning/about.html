Tentang Be-Py

Be-Py merupakan sebuah aplikasi berbasis web untuk belajar programming dasar dengan menggunakan bahasa Python. Be-Py menyediakan berbagai materi pembelajaran Python dasar yang disertai dengan contoh penggunaan sintaksnya. Setelah membaca materi, Anda diharuskan untuk mengerjakan sebuah soal coding yang jawabannya langsung diperiksa oleh Be-Py.
